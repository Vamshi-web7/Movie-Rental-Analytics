# Movie Rental Analytics

## Overview

This project analyzes the Sakila movie rental database to identify customer behavior, rental trends, and revenue insights using SQL, Excel, and Power BI.

## Tools Used

* SQL
* Power BI
* Excel

## Features

* Customer behavior analysis
* Revenue trend analysis
* KPI dashboard creation
* Rental trend analysis
* Data cleaning and validation

## SQL Concepts Used

* Joins
* Aggregations
* Window Functions
* Subqueries

## Dashboard Insights

* Top rented movie categories
* Monthly revenue analysis
* Customer activity tracking
* Rental performance KPIs

## Files Included

* SQL query files
* Dashboard files
* Dataset files
* Screenshots
* Documentation and reports
